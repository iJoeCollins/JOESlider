Setup
-----

Open your terminal and run:

> ```git clone git@github.com:iJoeCollins/JOESlider.git```

Drag the JOESlider folder into your project.

Then in your .pch file:

> ```#import "JOESlider.h"```

Or just import individual header files where needed.
